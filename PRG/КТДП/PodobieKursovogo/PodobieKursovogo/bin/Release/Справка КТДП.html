ОПИСАНИЕ ПРОГРАММЫ

    ЧТОБЫ ПОЛЬЗОВАТЕЛЬ СМОГ СУЩЕСТВЕННО УСКОРИТЬ НАБОР ТЕКСТА БЫЛО СОЗДАНО МНОЖЕСТВО СПЕЦИАЛИЗИРОВАННЫХ ПРОГРАММ, КОТОРЫЕ НАЗЫВАЮТСЯ КЛАВИАТУРНЫМИ ТРЕНАЖЕРАМИ. В ЧАСТНОСТИ, НАВЫК БЫСТРОГО НАБОРА ДАННЫХ НЕОБХОДИМ СПЕЦИАЛИСТАМ, ЧЬЯ ПРОФЕССИЯ НЕПОСРЕДСТВЕННО СВЯЗАНА С РАБОТОЙ НА КОМПЬЮТЕРЕ. ТАКОЙ ПРОФЕССИЕЙ ЯВЛЯЕТСЯ ПРОФЕССИЯ ПРОГРАММИСТА. НО, ЧТОБЫ ПРОГРАММИСТУ ЗНАЧИТЕЛЬНО УСКОРИТЬ ПРОЦЕСС НАПИСАНИЯ ПРОГРАММ, НЕДОСТАТОЧНО ТРЕНИРОВАТЬСЯ В ПРОГРАММАХ, НАПРАВЛЕННЫХ НА УСКОРЕНИЕ ПЕЧАТИ СПЛОШНЫХ ТЕКСТОВ НА РУССКОЙ РАСКЛАДКЕ КЛАВИАТУРЫ.

ФУНКЦИОНАЛЬНЫМ НАЗНАЧЕНИЕМ ПРОГРАММЫ ЯВЛЯЕТСЯ ВОЗМОЖНОСТЬ СОВЕРШЕНСТВОВАТЬ СКОРОСТЬ НАБОРА КЛЮЧЕВЫХ ВЫРАЖЕНИЙ ОПРЕДЕЛЕННЫХ ЯЗЫКОВ ПРОГРАММИРОВАНИЯ. КОНЕЧНЫМИ ПОЛЬЗОВАТЕЛЯМИ ЯВЛЯЮТСЯ ПРОГРАММИСТЫ, КОТОРЫЕ ВЕДУТ РАЗРАБОТКУ НА ОДНОМ ИЗ ТАКИХ ЯП, КАК C#, PASCAL, PYTHON, JAVASCRIPT, PL/SQL.

ПРИКЛАДНАЯ ПРОГРАММА «КЛАВИАТУРНЫЙ ТРЕНАЖЕР ДЛЯ ПРОГРАММИСТА» РАЗРАБОТАНА НА КУРСОВОЕ ПРОЕКТИРОВАНИЕ ПО ДИСЦИПЛИНЕ «КОНСТРУИРОВАНИЕ ПРОГРАММ И ЯЗЫКИ ПРОГРАММИРОВАНИЯ» ПОД РУКОВОДСТВОМ КОШЕЛЕВОЙ ВАЛЕНТИНЫ ПЕТРОВНЫ.

 

 

 

 

 

 

 

 


РАБОТА С ПРОГРАММОЙ

ДЛЯ ТОГО, ЧТОБЫ НАЧАТЬ ВЫПОЛНЯТЬ ЗАДАНИЕ, НЕОБХОДИМО ОТКРЫТЬ МЕНЮ, НАЖАВ НА ВЫЕЗЖАЮЩЕЕ МЕНЮ, НАХОДЯЩЕЕСЯ В ЛЕВОЙ СТОРОНЕ ПРОГРАММЕ.

НЕОБХОДИМО ВЫБРАТЬ ЯЗЫК ПРОГРАММИРОВАНИЯ, НА КОТОРОМ БУДУТ ВЫПОЛНЯТЬСЯ ЗАДАНИЯ.

ДАЛЕЕ НУЖНО ВЫБРАТЬ НОМЕР ЗАДАНИЯ, КОТОРОЕ БУДЕТ ОТКРЫТО ДЛЯ ВЫПОЛНЕНИЯ. ЗАДАНИЕ ПОД НОМЕРОМ 1 ПРЕДСТАВЛЯЕТ СОБОЙ ЗАРЕЗЕРВИРОВАННЫЕ СЛОВА, СОСТОЯЩИЕ ИЗ МИНИМАЛЬНОГО КОЛИЧЕСТВА СИМВОЛОВ, ЗАДАНИЕ ПОД НОМЕРОМ 5 СОСТОИТ ИЗ СЛОВ С МАКСИМАЛЬНЫМ КОЛИЧЕСТВОМ СИМВОЛОВ СООТВЕТСТВЕННО.

ПОСЛЕ ПРОЧТЕНИЯ ОКНА С ПРАВИЛАМИ ПРОХОЖДЕНИЯ НАЧНЕТСЯ ОТЧЕТ ТАЙМЕРА И БУДЕТ ДОСТУПНО ПРОХОЖДЕНИЕ ЗАДАНИЯ. ПЕРЕД НАЧАЛОМ ПРОХОЖДЕНИЯ НЕОБХОДИМО ПЕРЕКЛЮЧИТЬ РАСКЛАДКУ НА АНГЛИЙСКИЙ ЯЗЫК.

ПО ОКОНЧАНИЮ ПРОХОЖДЕНИЯ ПОЯВИТСЯ ОКНО С РЕЗУЛЬТАТАМИ, ПОКАЗЫВАЮЩИЕ КОЛИЧЕСТВО НАПЕЧАТАННЫХ СИМВОЛОВ, КОЛИЧЕСТВО ОШИБОК, ВРЕМЯ ВЫПОЛНЕНИЯ ЗАДАНИЯ И СКОРОСТЬ ПЕЧАТИ.
